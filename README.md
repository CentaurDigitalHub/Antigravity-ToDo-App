# Elegant To-Do List

A simple, premium To-Do List application built with plain HTML, CSS, and JavaScript.

## Features

-   **Add Tasks**: Easily add new tasks to your list.
-   **Mark as Complete**: Click any task to mark it as done with a satisfying strikethrough effect.
-   **Delete Tasks**: Remove completed tasks using the delete button (trash icon).
-   **Elegant Black Theme**: A sleek, modern dark mode design with glassmorphism effects and subtle animations.

## How to Run

1.  Navigate to the project folder.
2.  Locate the `index.html` file.
3.  Double-click `index.html` to open the application in your default web browser.

No installation or server is required!
